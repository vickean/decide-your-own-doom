# Combat enahncement for Markdoom
- the current Markdoom standard handles global varible pretty well but it does not have the ability to define and handle a battle event.
- Implement a battle system where the enemy units can be defined on a global level and instanced in a scene's battle. This allows for reuse as well as potentially scene specific variations based on the global definition.
- Behavior of the enemy units can also be defined in terms of what strategies it will use.
