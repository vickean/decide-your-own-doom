# Initial Groudwork
## Overview
- Goal: This app is a game engine for choose-your-own-adventure text-based games. The user feeds in a folder of markdown-like files that outline the branching story.
- This app should be built using React. Any design system is fine ie. shadcn.
- As part of this project, we should come up with a markdown-like format to describe scenes (nodes) of the branching storyline and the decisions that the user can choose. 
- An extension to that file format would be to support variables that can store data across the game. These variables can be used to store valus such as items or flags that tell the game that a player has done something.

Feel free to ask any questions to clarify details.
